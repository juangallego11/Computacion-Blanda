Percetron multicapa para calcular la función sinusoidal utilizando la API de Pybrain

Este trabajo tiene como objetivo implementar un Perceptrón Multicapa (MLP) de Red Neural Artificial
para conocer el valor correcto del seno de un ángulo entre 0 y 2pi, para ello se utilizo PyBrain que es una API para el desarrollo de una Red Neuronal Artificial.
Posteriormente, se generaron datos de entrenamiento y prueba para generar el aprendizaje del seno y, finalmente, se generaron gráficos con datos estadísticos sobre los insumos y aprendizajes de la red.
